Today marks the official release of **LaunchFlow V1.0.0**.

The manual submission grind is officially dead. This release brings our core automation engine out of private beta and into the hands of indie makers worldwide.

### New Features

- **The Automation Engine**: Our Playwright-based bots now support full auto-submission to Tier 1 directories like Uneed, DevHunt, and BetaList.
- **AI Launch Toolkit**: Built-in OpenAI integration to automatically generate product descriptions, Reddit posts, and Hacker News copy from a simple project URL.
- **Pay-Per-Launch Credits**: No recurring subscriptions. Buy credits, use them anytime to launch projects.
- **Live Submission Dashboard**: Watch in real-time as your project gets queued, submitted, and approved across the web.

### Improvements

- Completely redesigned the web app using the "Swiss-Punk" aesthetic for a faster, bolder experience.
- Migrated the frontend to Next.js App Router for 3x faster page loads.

*Go launch something awesome.*
